# Исследования конкурентов. MVP карточки wildberries

Площадка: Wildberries
Клиенты: продавцы с 100-500 SKU, ~20к селлеров

MPSTATS
selsup

1. Взять категорию. Например, обувь
2. Спарсить первые 100 лотов, их изображения.

Парсинг первых 100 лотов по категории (обувь) через WB API/Webbee/SellerMoon (1–5 мин/запрос) даёт изображения, отзывы, рейтинг, выручку.

Фильтр эффективности: Рейтинг >4.6 + отзывы >100 + выручка >500к руб/мес + конверсия >5% (если доступно). Исключите: отзывы <24ч после заказа, повторяющиеся тексты/эмодзи, >50% 5★ без фото.

Анализ шаблонов: Кластеризуйте изображения (OpenCV/Weaviate: цвета, текст via Tesseract, элементы: "до/после", размеры, шрифты). Топ-шаблоны: 7–8 фото + инфографика (схемы, сравнения).

| Метрика   | Фильтр успеха            | Исключить (фейк/изменения)            |
| --------- | ------------------------ | ------------------------------------- |
| Рейтинг   | >4.6/5 (>80% 5★)         | <4.5 или >95% 5★ без фото vc​         |
| Отзывы    | >100, фото >30%          | <24ч после заказа, дубли/эмодзи vk​   |
| Выручка   | >1млн руб/мес стабильная | Пиковые всплески >200% onegin-expert​ |
| Конверсия | >5% (если API)           | Нет данных — пропустить               |
| Доп       | Топ-10 поиска WB         | Новинки (<1мес) reklama.tochka​       |
