# Категории товаров

1 уровень категорий:
- Бытовая электроника (тесты на бытовых пылесосах)
- Аксессуары для гаджетов
- Одежда и аксессуары (особенно женская мода)
- Товары для дома и кухни
- Канцелярские принадлежности
- Товары для детей, игрушки
- Косметика и товары для красоты
- Спорттовары, фитнес-аксессуары, товары для здоровья
- Пряжа, аксессуары для рукоделия, бижутерия

2 уровень категорий:
- товары для домашних животных (грызуны, рептилии)
- эко-товары и локальные фермерские продукты
- сезонные и тематические товары
- Товары для хобби, рукоделия, моделирование
- Медицинские товары (кроме лекарств)
- проф.инструменты
- промышленное оборудование

На маркетплейсах:
- качество дизайна в топе

На фриланс биржах:
- скорость закрытия заказов
- число открытых заказов

Платформы анализа спроса на маркетплейсах
Moneyplace, MPStats, SellMonitor, MarketGuru, LikeStats
