## ⚡ Реальные сценарии использования

### 🟢 **Вам подходит локальная генерация, если:**

| Сценарий | Почему локально |
|----------|----------------|
| 🔐 Работа с конфиденциальными данными | Приватность > экономия |
| 🔄 Частые итерации / A/B тесты | Нет платы за «неудачные» генерации |
| 🧩 Кастомные воркфлоу (ComfyUI, LoRA) | Полная свобода настройки |
| 🌙 Генерация по расписанию (ночью) | Используете «бесплатное» время сервера |
| 📦 Пакетная обработка тысяч файлов | Фиксированная цена независимо от объёма |

### 🔴 **Вам лучше облако, если:**

| Сценарий | Почему облако |
|----------|---------------|
| 🧪 Редкие запросы (<10% загрузки сервера) | Платите только за использование |
| 🚀 Нужна максимальная скорость | Облачные кластеры быстрее одиночной 3090 |
| 🌍 Доступ с разных устройств | Нет привязки к домашнему серверу |
| 🛠️ Не хотите заниматься поддержкой | Нет апдейтов, драйверов, охлаждения |

## 📊 Сравнение по трём сценариям

### **1️⃣ ТЕКСТ (LLM) — Claude Sonnet vs Llama 3.3 70B локально**

| Параметр | Локально (1×3090) | Claude Sonnet API |
|----------|-------------------|-------------------|
| **Скорость** | 20-35 ток/с (70B, Q4) | 50-100+ ток/с |
| **Цена за 1М output** | ~2 060 ₽* | $15 ≈ 1 350 ₽ [[2]] |
| **Контекст** | до 32K (с потерей скорости) | до 200K нативно |
| **Качество** | ~90% от Sonnet | 🏆 State-of-the-art |
| **Приватность** | ✅ Полная | ❌ Данные у провайдера |
| **Лимиты** | ❌ Нет | ✅ Есть (rate limits) |

*\*Расчёт: 32 000 ₽ / 15,55 млн токенов = 2,06 ₽/1000 токенов*

**🎯 Когда локально выгоднее:**
- >25 млн токенов/мес (~10 ток/с непрерывно)
- Нужна полная приватность данных
- Работа с чувствительной информацией

### **2️⃣ ИЗОБРАЖЕНИЯ — FLUX.1 Dev локально vs Fal.ai/Replicate**

| Параметр | Локально (1×3090) | Fal.ai / Replicate |
|----------|-------------------|-------------------|
| **Скорость (1080p)** | 45-50 сек/изображение [[30]] | 15-25 сек/изображение |
| **Цена за изображение** | ~2,06 ₽* | $0.025 ≈ 2,25 ₽ [[30]] |
| **Качество** | Идентичное (та же модель) | Идентичное |
| **Кастомизация** | ✅ Любые LoRA, ControlNet | ❌ Ограничено |
| **Пакетная генерация** | ✅ Бесплатно | ❌ Плата за каждое |

*\*32 000 ₽ / (86 400 сек / 47 сек) ≈ 2,06 ₽ за изображение при 24/7 работе*

**🎯 Когда локально выгоднее:**
- >14 000 изображений/мес (~470 в день)
- Нужны кастомные воркфлоу (ComfyUI)
- Генерация в реальном времени без задержек API

### **3️⃣ ВИДЕО — Wan 2.2 локально vs Fal.ai/Runway**

| Параметр | Локально (1×3090) | Fal.ai (Wan 2.2/Veo) |
|----------|-------------------|---------------------|
| **Скорость (1080p@24fps, 5 сек)** | 15-25 минут | 2-5 минут |
| **Цена за 1 сек видео** | ~0,43 ₽* | $0.20-0.40 ≈ 18-36 ₽ [[55]][[56]] |
| **Качество** | Идентичное (та же модель) | Идентичное |
| **Доступность** | ✅ Всегда | ❌ Может быть очередь |
| **Пост-обработка** | ✅ Полная | ❌ Ограничена |

*\*32 000 ₽ / (86 400 сек / 1 200 сек на 5-сек видео) ≈ 0,43 ₽/сек при 24/7*

**🎯 Когда локально выгоднее:**
- >900 секунд видео/мес (~30 сек/день)
- Эксперименты с параметрами (дешевле итерации)
- Долгие рендеры без таймаутов облака
