# Cardgen (ветка extgen) — только RouterAI

## Идея

Вся генерация и проверка — **удалённо** через **[RouterAI](https://routerai.ru)** (OpenAI-совместимый API). Локально нет CV/rembg.

### Основной сценарий: `generate`

1. Пользователь вводит **черновой промпт**.
2. **GPT-5.4** (`openai/gpt-5.4`) усиливает промпт: композиция, свет, детали; при необходимости явно описывает сегментацию сцены (изолированный товар, фон и т.д.).
3. **Nano Banana** (`google/gemini-2.5-flash-image`) **генерирует изображение по усиленному тексту** (text-to-image).
4. **GPT-5.4** смотрит на результат (vision), даёт оценку и **нумерованный список**, что улучшить в следующей итерации.
5. CLI печатает усиленный промпт, путь к PNG и блок ревью; опционально `--review-file` сохраняет ревью и промпт в файл.

Переопределение моделей: `CARDGEN_CHAT_MODEL`, `CARDGEN_IMAGE_MODEL`.

### Дополнительно: `process`

Папка с **исходными фото**: GPT по каждому кадру формирует инструкцию, Nano Banana **редактирует** изображение (image + текст). Для сценария «только текст из головы» используйте **`generate`**.

## Требования

- Python ≥ 3.10  
- **`ROUTERAI_API_KEY`** (или `OPENAI_API_KEY`) — см. ниже про `.env`  
- Опционально **`ROUTERAI_BASE_URL`** (по умолчанию `https://routerai.ru/api/v1`)

### Ключ API (не в git)

1. Скопируйте **`.env.example`** в **`.env`** в корне репозитория.  
2. Впишите в `.env` строку `ROUTERAI_API_KEY=ваш_токен`.  
3. Файл **`.env` уже в `.gitignore`** — в коммиты не попадает. Шаблон **`.env.example`** без секретов можно хранить в git.

При запуске CLI переменные из `.env` подхватываются автоматически (`python-dotenv`). Вручную задавать env каждый раз не нужно. Переменные в оболочке по-прежнему имеют приоритет над строками из `.env` (если задали и то и другое). Пустая строка в `.env` для опциональных ключей трактуется как «взять значение по умолчанию в коде».

## Установка

```powershell
cd d:\langs\python\cardgen
python -m venv .venv
.\.venv\Scripts\pip install -e .
copy .env.example .env
# Отредактируйте .env и вставьте ROUTERAI_API_KEY
```

## CLI

```powershell
# Ключ уже в .env; при необходимости можно временно: $env:ROUTERAI_API_KEY = "..."

# Главный сценарий: усиление -> генерация -> ревью
python -m cardgen generate --prompt "беспроводные наушники на белом фоне, студийный свет" --context "карточка WB, главное фото"

# Без усиления или без ревью
python -m cardgen generate -p "закат в горах" --skip-enhance
python -m cardgen generate -p "..." --skip-review

# Сохранить ревью в файл
python -m cardgen generate -p "..." -o samples/out/card.png --review-file samples/out/review.md

# Старая цепочка: папка in -> правки по фото
python -m cardgen process --in samples/in --out samples/out
```
