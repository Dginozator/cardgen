# Cardgen (`extgen`)

Эта ветка — **старая экспериментальная линия** (Python / ранний прототип). Актуальная разработка приложения ведётся в **`main`**.

## Repository template (`template/v1.0.0`)

Чтобы создать новый репозиторий от зафиксированного снимка-шаблона (тег **`template/v1.0.0`** на GitHub), выполните:

```bash
git clone https://github.com/Dginozator/cardgen.git
cd cardgen
git checkout template/v1.0.0
```

Тег указывает на конкретный коммит и не двигается при новых push в `main` — это база для форков и новых проектов. Свежий код — в ветке **`main`**.
